<summary><sub>how this chord sheet is made</sub></summary>
<br>
<div align="left">

Each box is a **month**; the six strings are **Mon–Sat**, the fret columns are the
**weeks** of that month. A filled dot is a day I shipped code (bigger = more), the
brass ring is that month's busiest day (the chord's *root*), and the `o`/`x` above
the nut is whether I worked that **Sunday**. Streaks are drawn as **slur** arcs, and
the outlined box with the ◆ is my **home chord** — the month I contributed most.

It’s generated daily from my live GitHub contribution calendar by a small GitHub
Action and rendered as a JS-free animated SVG (dark + light). No snakes were harmed.
